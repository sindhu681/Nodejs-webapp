## 🛠️ Prerequisites
Before running the application, ensure you have:
- **Node.js** (v14+ recommended) → [Download](https://nodejs.org/)
- **VS Code** (or any code editor)
- **Live Server** extension for frontend (Optional)
- **Git** (for version control)

commands: 
mkdir node-backend
cd node-backend
npm install
npm init -y : It will create package.json file
npm install express cors: added 71 packages, and audited 72 packages in 5s
node server.js
